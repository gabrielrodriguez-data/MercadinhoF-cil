Objetivo do Software

O sistema Mercadinho Fácil tem como objetivo principal automatizar a rotina de um estabelecimento comercial de pequeno porte, permitindo o gerenciamento eficiente de vendas, controle de estoque, movimentação de caixa, cadastro de clientes, fornecedores, produtos e usuários por meio de uma aplicação desktop robusta desenvolvida em Java.

Funcionalidades do Sistema

Gerenciamento de Clientes para cadastrar, atualizar e consultar informações dos clientes do estabelecimento.

Gerenciamento de Fornecedores para registrar e controlar os dados dos fornecedores de mercadorias.

Gerenciamento de Produtos para o controle completo de cadastro, preços e itens disponíveis para comercialização.

Controle de Estoque para acompanhar as entradas, saídas e a quantidade atualizada de mercadorias.

Registro de Vendas para efetuar o atendimento ao cliente, calcular os itens comercializados e finalizar as transações.

Movimentação de Caixa para controlar as entradas, as saídas e o saldo financeiro diário do caixa.

Gestão de Usuários para o controle de acesso, perfis e segurança ao sistema.

Painel Dashboard para visualização rápida de indicadores e gráficos gerenciais do negócio.
